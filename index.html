MCP Servers

Model Context Protocol servers for macOS and Apple ecosystem integrations.

  • multimodal-imessage-mcp — iMessage access with conversation search and attachment handling
  • icloud-calendar-mcp — iCloud Calendar event management with full CRUD via local macOS
  • reminders-mcp — Apple Reminders integration with full list and reminder management
  • apple-notes-mcp — Comprehensive Apple Notes server for local macOS note management
  • safari-mcp — Full Safari browser control, page extraction, and WebDriver-backed screenshots
  • gmail-multi-inbox-mcp — Multi-account Gmail access with built-in OAuth onboarding
  • app-store-connect-mcp — App Store Connect APIs and release automation

Install

Most CLI tools are available via Homebrew:

brew tap tszaks/tap
brew install <tool-name>

MCP servers can be added to Claude Code or any MCP-compatible client via npx or local install. See each repo for setup instructions.

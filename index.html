MCP Servers

Model Context Protocol servers for macOS and Apple ecosystem integrations.

  • multimodal-imessage-mcp iMessage access with conversation search and attachment handling mcp
  • icloud-calendar-mcp iCloud Calendar event management with full CRUD via local macOS mcp
  • reminders-mcp Apple Reminders integration — list and reminder management mcp
  • safari-mcp Full Safari browser control, page extraction, and WebDriver screenshots mcp
  • gmail-multi-inbox-mcp Multi-account Gmail access with built-in OAuth onboarding mcp
  • app-store-connect-mcp App Store Connect APIs and release automation mcp

Install

Most CLI tools are available via Homebrew:

brew tap tszaks/tap
brew install <tool-name>

MCP servers can be added directly to Claude Code or any MCP-compatible client via npx or local install.
